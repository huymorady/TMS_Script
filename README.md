# TMS_Script

tms.skyunion.net 웹 CAT 툴용 Tampermonkey 단축키 스크립트 모음입니다.

마우스 우클릭 메뉴를 거쳐야 했던 반복 작업들을 키보드 단축키로 대체하여 번역 작업 효율을 높여줍니다.

---

## 스크립트 목록

| 파일 | 설명 |
|---|---|
| `cat-tool-shortcuts.user.js` | 메인 단축키 스크립트 |
| `cat-tool-lookup.user.js` | 번역 조회 팝업 (마크다운 테이블 매칭) |
| `cat-tool-tb.user.js` | TB 도구 (용어 목록 추출 + 검수) |
| `nara-speller-auto.user.js` | 나라 맞춤법 검사기 자동 붙여넣기 보조 스크립트 |

---

## 설치 방법

### 사전 준비

1. Chrome 브라우저에 [Tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) 확장 프로그램을 설치합니다.
2. Chrome 퍼즐 아이콘 → Tampermonkey → **"이 사이트에서 데이터 읽기 및 변경 가능"**이 허용되어 있는지 확인합니다.

### 스크립트 설치

아래 링크를 브라우저에서 열면 Tampermonkey 설치 화면이 자동으로 나타납니다.

- **메인 단축키**: [cat-tool-shortcuts.user.js](https://raw.githubusercontent.com/huymorady/TMS_Script/main/cat-tool-shortcuts.user.js)
- **번역 조회 팝업**: [cat-tool-lookup.user.js](https://raw.githubusercontent.com/huymorady/TMS_Script/main/cat-tool-lookup.user.js)
- **TB 도구**: [cat-tool-tb.user.js](https://raw.githubusercontent.com/huymorady/TMS_Script/main/cat-tool-tb.user.js)
- **맞춤법 검사기 보조**: [nara-speller-auto.user.js](https://raw.githubusercontent.com/huymorady/TMS_Script/main/nara-speller-auto.user.js)

### 업데이트

스크립트에 `@updateURL`이 설정되어 있어 Tampermonkey가 자동으로 업데이트를 감지합니다. 즉시 업데이트가 필요하면 Tampermonkey 대시보드 → 해당 스크립트 → 설정 탭 → **"업데이트 확인"**을 클릭하세요.

## 단축키 총 정리

### 메인 단축키 스크립트

#### TB 추가 — `Alt + ``

원문 텍스트를 드래그한 상태에서 사용합니다.

- 우클릭 메뉴의 "添加术语表"를 자동으로 실행하여 TB 추가 팝업을 엽니다.
- 팝업의 세 필드가 자동으로 채워집니다:
  - **术语备注** → `용어` (고정, 필요 시 수정 가능)
  - **Chinese Simplified** → 드래그한 원문 (사이트 오류로 빈칸일 때도 자동 보완)
  - **Korean** → 같은 행의 번역문 (입력되어 있는 경우)

#### TM 검색 — `Alt + 1` (드래그 상태)

- **원문에서 드래그** → 全局搜索 탭에서 `zh-Hans->ko` 방향으로 검색
- **번역문에서 드래그** → 全局搜索 탭에서 `ko->zh-Hans` 방향으로 자동 전환 후 검색
- 검색어가 자동 입력되고 검색 버튼까지 클릭됩니다.

#### 사전 검색 — `Alt + 2~4` (드래그 상태)

원문 또는 번역문에서 텍스트를 드래그한 상태에서 사용합니다.

- `Alt + 2` → 네이버 중국어 사전 검색
- `Alt + 3` → 네이버 영어 사전 검색
- `Alt + 4` → 네이버 한국어 사전 검색

#### CAT 목록 TM 적용 — `Alt + 1~6` (드래그 없이)

번역 입력창에 포커스된 상태에서 텍스트 드래그 없이 사용합니다.

- 오른쪽 CAT 목록의 N번째 항목의 체크(✓) 버튼을 클릭합니다.
- `Alt+1`은 1번(최상위 매치), `Alt+2`는 2번, 최대 6번까지 지원합니다.

#### 태그 삽입 — `Alt + T`

번역 입력창에 포커스된 상태에서 사용합니다.

- 같은 행의 원문에서 태그를 자동으로 추출하여, 누를 때마다 원문 순서대로 하나씩 커서 위치에 삽입합니다.
- 다른 세그먼트로 이동하면 자동으로 리셋됩니다.
- 번역문에 이미 존재하는 태그는 건너뛰고 다음 태그로 넘어갑니다.
- 지원 태그 형식: `{0}`, `{text1}`, `{player_name}`, `%s`, `%d`, `%1$s`, `<br/>`, `<color=red>`, `</color>`, `\n`, `[color=#FF0000]` 등

#### 맞춤법 검사 — `Alt + S` / `Alt + Shift + S`

- `Alt + S` → 현재 세그먼트의 번역문을 클립보드에 복사하고, 나라 맞춤법 검사기를 새 탭으로 엽니다.
- `Alt + Shift + S` → 전체 세그먼트의 번역문을 줄바꿈으로 합쳐서 클립보드에 복사하고, 맞춤법 검사기를 엽니다.
- `nara-speller-auto.user.js`가 설치되어 있으면 자동으로 붙여넣기 + 검사 실행까지 진행됩니다.

#### 저장 개선 — `Ctrl + Enter`

- IME(한국어 입력기) 조합 중인 마지막 글자를 확정한 뒤 저장합니다. 마지막 글자가 잘리는 버그를 방지합니다.
- 각 줄 끝의 불필요한 공백(스페이스)을 자동으로 제거합니다.

#### 도움말 — `Alt + H`

- 모든 스크립트의 단축키를 한눈에 볼 수 있는 도움말 팝업을 엽니다.

---

### 번역 조회 스크립트

#### 팝업 열기/닫기 — `Alt + Q`

- 마크다운 테이블 데이터를 붙여넣고 파싱하면 원문-번역문 매칭 데이터가 저장됩니다.
- 헤더에 "원문"과 "번역문" 열 이름이 있어야 자동으로 해당 열을 감지합니다.
- 여러 번 붙여넣기해서 데이터를 누적할 수 있습니다.
- `sessionStorage`를 사용하여 새로고침해도 데이터가 유지됩니다. 탭을 닫으면 자동 삭제됩니다.

#### 현재 세그먼트 매칭 삽입 — `Alt + W`

- 현재 포커스된 세그먼트의 원문을 저장된 데이터에서 찾아 번역문을 자동 삽입합니다.
- `<br>` ↔ 줄바꿈 변환에 대응하는 정규화 매칭을 지원합니다.
- 삽입 시 번역문의 `<br>`은 실제 줄바꿈(`\n`)으로 변환됩니다.

#### 전체 세그먼트 일괄 삽입 — `Alt + Shift + W`

- 화면의 전체 세그먼트를 일괄 매칭해서 삽입합니다.
- 이미 번역문이 입력된 세그먼트는 건너뜁니다.

---

### TB 도구 스크립트

#### TB 도구 팝업 열기/닫기 — `Alt + B`

팝업에는 두 개의 탭이 있습니다.

**목록 탭**
- 현재 파일의 모든 세그먼트에서 TB 용어를 자동 추출합니다.
- 원문 기준 정렬된 테이블로 표시됩니다.
- 📋 버튼으로 탭 구분 형식으로 클립보드 복사 (엑셀 붙여넣기 가능)

**검수 탭**
- 각 세그먼트의 번역문에 TB 번역어가 올바르게 반영되었는지 검사합니다.
- 미반영 항목을 세그먼트 번호, 원문 용어, TB 번역어와 함께 표시합니다.
- 항목 클릭 시:
  - 해당 세그먼트로 자동 스크롤 이동 + 번역 입력창 포커스
  - TB 번역어를 클립보드에 자동 복사 → `Ctrl+V`로 바로 붙여넣기 가능
- 📋 버튼으로 검수 결과를 클립보드에 복사

## 팝업 공통 기능

모든 팝업(번역 조회, TB 도구, 도움말)에 공통으로 적용됩니다.

- **헤더 제목 클릭** → 접기/펼치기 (헤더만 남기고 본문 숨김)
- **헤더 드래그** → 팝업 위치 자유 이동
- **✕ 버튼** → 팝업 닫기

## 기술 정보

- **캡처 단계 이벤트 처리**: 웹 CAT 툴이 키 이벤트를 가로채지 못하도록 캡처 단계(`addEventListener`의 세 번째 인자 `true`)에서 먼저 실행됩니다.
- **`@run-at document-start`**: 메인 단축키 스크립트는 페이지 스크립트보다 먼저 로드되어 안정적으로 동작합니다.
- **Vue/React 반응성 대응**: `nativeSetter`를 사용하여 프레임워크가 값 변경을 인식하도록 합니다.
- **sessionStorage**: 번역 조회 데이터는 새로고침 시 유지되며, 탭 닫기 시 자동 삭제됩니다.
- **자동 업데이트**: `@updateURL` / `@downloadURL`이 GitHub Raw URL로 설정되어 있어, `@version`을 올려 push하면 Tampermonkey가 자동으로 업데이트를 감지합니다.

## 문제 해결

**스크립트가 동작하지 않는 경우**

1. Tampermonkey 아이콘에 숫자가 표시되는지 확인합니다. 표시되지 않으면 Chrome 확장 프로그램 권한 설정에서 Tampermonkey의 사이트 접근을 허용해 주세요.
2. F12 → 콘솔 탭에서 `[CAT 단축키]`, `[번역 조회]`, `[TB 도구]`로 시작하는 로그 메시지가 있는지 확인합니다.

**맞춤법 검사기에서 자동 붙여넣기가 안 되는 경우**

- 브라우저에서 클립보드 접근 권한을 요청할 수 있습니다. "허용"을 눌러주세요.

**동료 컴퓨터에 설치할 때**

- 스크립트 설치 링크를 브라우저에서 열면 Tampermonkey 설치 화면이 자동으로 나타납니다.
- Chrome 확장 프로그램 권한에서 Tampermonkey의 사이트 접근이 허용되어 있는지 반드시 확인해 주세요.
