# TMS_Script

tms.skyunion.net 웹 CAT 툴용 Tampermonkey 단축키 스크립트 모음입니다.

마우스 우클릭 메뉴를 거쳐야 했던 반복 작업들을 키보드 단축키로 대체하여 번역 작업 효율을 높여줍니다.

---

## 스크립트 목록

| 파일 | 설명 |
|---|---|
| `cat-tool-shortcuts.user.js` | 메인 단축키 스크립트 |
| `nara-speller-auto.user.js` | 나라 맞춤법 검사기 자동 붙여넣기 보조 스크립트 |

---

## 설치 방법

### 사전 준비

1. Chrome 브라우저에 [Tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) 확장 프로그램을 설치합니다.
2. Chrome 퍼즐 아이콘 → Tampermonkey → **"이 사이트에서 데이터 읽기 및 변경 가능"**이 허용되어 있는지 확인합니다.

### 스크립트 설치

아래 링크를 브라우저에서 열면 Tampermonkey 설치 화면이 자동으로 나타납니다.

- **메인 단축키**: [cat-tool-shortcuts.user.js](https://raw.githubusercontent.com/huymorady/TMS_Script/main/cat-tool-shortcuts.user.js)
- **맞춤법 검사기 보조**: [nara-speller-auto.user.js](https://raw.githubusercontent.com/huymorady/TMS_Script/main/nara-speller-auto.user.js)

두 스크립트 모두 설치해야 맞춤법 검사 기능이 완전하게 동작합니다.

### 업데이트

스크립트에 `@updateURL`이 설정되어 있어 Tampermonkey가 자동으로 업데이트를 감지합니다. 즉시 업데이트가 필요하면 Tampermonkey 대시보드 → 해당 스크립트 → 설정 탭 → **"업데이트 확인"**을 클릭하세요.

## 단축키 목록

### TB 추가 — `Alt + ``

원문 텍스트를 드래그한 상태에서 사용합니다.

- 우클릭 메뉴의 "添加术语表"를 자동으로 실행하여 TB 추가 팝업을 엽니다.
- 같은 행에 번역문이 입력되어 있으면 **Korean 필드에 자동 입력**됩니다.
- **术语备注** 필드에는 `용어`가 기본값으로 입력됩니다. 필요 시 수정 가능합니다.

### TM 검색 — `Alt + 1~6` (드래그 상태)

원문 텍스트를 드래그한 상태에서 사용합니다.

- 우클릭 메뉴의 "搜索记忆库"를 자동으로 실행합니다.
- 어떤 숫자(1~6)를 눌러도 동일하게 TM 검색이 실행됩니다.

### CAT 목록 TM 적용 — `Alt + 1~6` (드래그 없이)

번역 입력창에 포커스된 상태에서 텍스트 드래그 없이 사용합니다.

- 오른쪽 CAT 목록의 N번째 항목의 체크(✓) 버튼을 클릭합니다.
- `Alt+1`은 1번(최상위 매치), `Alt+2`는 2번, 최대 6번까지 지원합니다.

### 태그 삽입 — `Alt + T`

번역 입력창에 포커스된 상태에서 사용합니다.

- 같은 행의 원문에서 태그를 자동으로 추출하여, 누를 때마다 **원문 순서대로 하나씩** 커서 위치에 삽입합니다.
- 다른 세그먼트로 이동하면 자동으로 리셋됩니다.
- 번역문에 **이미 존재하는 태그는 건너뛰고** 다음 태그로 넘어갑니다.
- 지원 태그 형식: `{0}`, `{text1}`, `%s`, `%d`, `%1$s`, `<br/>`, `<color=red>`, `</color>`, `\n`, `[color=#FF0000]` 등

### 맞춤법 검사 (현재 세그먼트) — `Alt + S`

번역 입력창에 포커스된 상태에서 사용합니다.

- 현재 세그먼트의 번역문을 **클립보드에 복사**하고, 나라 맞춤법 검사기를 새 탭으로 엽니다.
- `nara-speller-auto.user.js`가 설치되어 있으면 **자동으로 붙여넣기 + 검사 실행**까지 진행됩니다.

### 맞춤법 검사 (전체 세그먼트) — `Alt + Shift + S`

아무 상태에서나 사용 가능합니다.

- 현재 파일의 **모든 세그먼트 번역문**을 줄바꿈으로 합쳐서 클립보드에 복사하고, 나라 맞춤법 검사기를 새 탭으로 엽니다.
- 마찬가지로 보조 스크립트가 설치되어 있으면 자동 붙여넣기 + 검사가 실행됩니다.

| `Alt + Shift + S` | - | 전체 세그먼트 맞춤법 검사 |

## 기술 정보

- **캡처 단계 이벤트 처리**: 웹 CAT 툴이 키 이벤트를 가로채지 못하도록 캡처 단계(`addEventListener`의 세 번째 인자 `true`)에서 먼저 실행됩니다.
- **`@run-at document-start`**: 페이지 스크립트보다 먼저 로드되어 안정적으로 동작합니다.
- **자동 업데이트**: `@updateURL` / `@downloadURL`이 GitHub Raw URL로 설정되어 있어, `@version`을 올려 push하면 Tampermonkey가 자동으로 업데이트를 감지합니다.

## 문제 해결

**스크립트가 동작하지 않는 경우**

1. Tampermonkey 아이콘에 숫자 `1`(또는 `2`)이 표시되는지 확인합니다. 표시되지 않으면 Chrome 확장 프로그램 권한 설정에서 Tampermonkey의 사이트 접근을 허용해 주세요.
2. F12 → 콘솔 탭에서 `[CAT 단축키]`로 시작하는 로그 메시지가 있는지 확인합니다.

**맞춤법 검사기에서 자동 붙여넣기가 안 되는 경우**

- 브라우저에서 클립보드 접근 권한을 요청할 수 있습니다. "허용"을 눌러주세요.
